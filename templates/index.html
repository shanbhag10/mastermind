MASTERMiND

Become the mastermind by decoding the hidden combination of colors.

  1. Guess the 4 color code

  2. Receive a result for your attempt

    - correct color, wrong position

    - correct color, correct position

  3. Use this result to make a better educated guess in the next round
  4. Try to successfully decrypt the code in minimum number of attempts
